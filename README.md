# Pole Finances

Pole Finances é sistema web para catalogar material de estudo e referências, send bem leve, pois utiliza apenas com HTML, CSS e JS.

O sistema que foi criado 100% com IA, traduzindo a necessidade do criador neste sistema que vos apresento, sem digitar uma linha de código.

Este permite você organizar seu catálogo de material de estudo e referências, separado em dois catálogo, um para português do Brasil e outro para inglês dos EUA.

Seu design foi voltado incialmente para finanças, uma necessidade atual do criador, mas é possível personalizar e abranger qualquer tipo de catálogo, para qualquer assunto.

Essa flexibilidade se dá por ele permitir adcionar itens ao catálogo, atualizar e remover, permitindo pesquisa robusta por simliridade e também ordenação.

Contudo, não contempla um banco de dados em nuvem, ficando os dados armazenados localmente, porém permite exportar estes dados para fazer backup ou até compartilhar com outras pessoas.

No caso de compartilhamento, conta com três atalhos para compartilhar o link via e-mail, Whatsapp e Telegram, ficando a seu critério enviar seu catálogo exportado.

Por fim, tem a opção de importar, que permite você aceitar ou não novos itens no seu catálogo bem como atulizar itens já existentes.
